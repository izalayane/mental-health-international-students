# 🎓 Mental Health of International Students

## 📊 Overview
This project analyzes whether studying abroad affects students' mental health using SQL and data from a Japanese university survey.

## 🧠 Objectives
- Analyze depression risk among international students
- Understand the impact of social connectedness
- Evaluate acculturative stress
- Explore the effect of length of stay

## 🛠 Tools Used
- SQL (PostgreSQL)
- DataCamp DataLab
- Python (Notebook environment)

## 📁 Files
- notebook.ipynb → analysis and queries
- students.csv → dataset
- mentalhealth.jpg → visualization / reference

## 📈 Key Insights
- International students showed higher mental health risks
- Social connectedness reduced depression indicators
- Acculturative stress increased mental health challenges

## 🚀 How to Run
1. Open the notebook.ipynb
2. Run cells step by step
